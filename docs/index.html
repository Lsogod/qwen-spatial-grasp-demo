典型流程

`Qwen` 模式下，模型负责动作规划和目标部件定位；`main_vlm.py` 负责把像素目标交给抓取候选生成、 约束筛选和机械臂执行。`svp` 模式下，再额外接入 `backend/` 中的 SAM + DINO 服务。 

Instruction -> Qwen Planner -> Part Grounding -> Pixel Target ->
Grasp Candidate Filtering -> UR5e Execution
